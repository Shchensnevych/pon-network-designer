🚀 Швидкий старт

  1. Додавання обладнання: Виберіть інструмент (OLT/FOB/ONU) зліва або перетягніть на карту
  2. З'єднання: Виберіть "Магістраль" або "Патчкорд", клікніть на початковий вузол, потім на кінцевий
  3. Налаштування: Клікніть на вузол → права панель покаже властивості (потужність, сплітери, гілки)
  4. Звіти: Кнопка "📊 Звіт / Експорт" → технічна частина + економічна для кошторису

📐 Основи проектування PON-мереж

1. Типи дільників та їх логіка

Мережа будується на комбінації двох типів сплітерів:

  • FBT (процентні): дві нерівні гілки. Гілка Y (магістраль, напр. 90%) — мінімальне затухання, передача сигналу на наступний вузол (FOB). Гілка X (відвід, напр. 10%) — більше затухання, підключення абонентів у поточному вузлі.
  • PLC (рівномірні): ділять сигнал порівну між усіма портами (1×4, 1×8, 1×16). Підключаються після гілки X дільника FBT.

2. Побудова магістралі (шина)

  • OLT → кабель → FBT (вхід).
  • FBT (гілка X / Tap) → PLC → кінцеві ONU (абоненти біля вузла).
  • FBT (гілка Y / Pass) → кабель → наступний FOB (вхід наступного FBT).

3. Розрахунок бюджету потужності

  • Система автоматично рахує затухання: сума затухань дільників + втрати в кабелі (напр. 0,4 дБ/км) + втрати на зварках (~0,05 дБ) + втрати на конекторах (~0,3–0,5 дБ).
  • Цільовий сигнал: від −16 дБ до −25 дБ.
  • Критичний поріг: нижче −27 дБ — сигнал занадто слабкий, ONU не запрацює.

4. Корисна порада

Завжди лишайте експлуатаційний запас (~2 дБ) на випадок майбутніх ремонтів або забруднення патчкордів.

📋 Детальна технічна інформація: відкрийте «📋 Довідка затухань» (кнопка в лівій панелі) — там таблиці затухань FBT/PLC, механічні втрати, оптичне волокно, пороги ONU/OLT та типова схема розрахунку.
